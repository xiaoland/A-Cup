╔═══════════════════════════════════════════════════════════════════════════╗
║                         DESIGN PRINCIPLES                                 ║
╚═══════════════════════════════════════════════════════════════════════════╝

1. SIMPLICITY
   └─ Minimal nesting, no unnecessary wrappers

2. CONSISTENCY
   └─ Same pattern across similar view types

3. UTILITY-FIRST
   └─ Use UnoCSS/Tailwind utilities over custom CSS

4. COMPONENT-NATIVE
   └─ Let PrimeVue components style themselves

5. RESPONSIVE
   └─ Mobile-first with breakpoint modifiers
